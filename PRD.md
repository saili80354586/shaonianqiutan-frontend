## 1. 项目概述

### 1.1 项目背景
**少年球探** 是一个专注于8-15岁青少年足球运动员的专业服务平台。通过专业比赛视频分析与球探报告服务，打造"视频采集→专业分析→内容传播→人才追踪→商业变现"的完整闭环。

### 1.2 核心团队
三位高资质分析师组合：
1. **亚足联A级教练员** - 青训标准制定、训练方向判断
2. **俄罗斯职业俱乐部留洋球员** - 实战视角、海外经验、天赋挖掘
3. **中超退役职业球员+亚足联B级教练员** - 国内青训体系、俱乐部资源对接

### 1.3 目标用户
**主要用户：** 8-15岁青少年足球运动员的家长

**核心痛点：**
- 认知痛点：看不懂孩子比赛表现
- 指导痛点：缺乏专业教练点评
- 展示痛点：优秀表现缺乏展示渠道
- 信任痛点：市场服务专业性参差不齐
- 记录痛点：缺乏系统性成长档案

### 1.4 技术栈
- **前端：** React 19 + TypeScript + Vite + Tailwind CSS v3.4 + React Router v7
- **后端：** Go + Gin + GORM + SQLite
- **部署：** 待定
- **第三方服务：** 高德地图API（地图）、微信支付/支付宝（支付）

## 6. API 接口规范（已定义）

### 6.1 认证模块 `/api/auth`

- `POST /send-code` - 发送短信验证码
- `POST /register` - 用户注册
- `POST /login` - 用户登录
- `POST /reset-password` - 重置密码
- `GET /me` - 获取当前用户信息
- `PUT /me` - 更新用户信息

### 6.2 报告模块 `/api/report`
- `GET /list` - 获取公开报告列表
- `GET /my` - 获取我的报告（购买的）
- `GET /published` - 获取我发布的报告（分析师）
- `GET /:id` - 获取报告详情
- `POST /` - 创建报告（分析师）
- `PUT /:id` - 更新报告
- `GET /:id/download` - 下载PDF报告

### 6.3 订单模块 `/api/order`
- `GET /` - 获取我的订单列表
- `POST /` - 创建订单
- `GET /:id` - 获取订单详情
- `POST /:id/payment` - 创建支付

### 6.4 其他模块
- 分析师申请、球探地图、用户资料等API已定义
